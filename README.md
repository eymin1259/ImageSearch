# ImageSearch
이미지 검색 앱 개발
- Clean-Architecture 구조를 적용하여, Presentation Layer, Domin Layer, Data Layer 분리
- ReactorKit을 적용하여 단방향 반응형 Swift 애플리케이션 아키텍처 구현
- Quick, Nimble을 사용하여 테스트 코드 작성

## [ Technical Skills ]
- Clean Architecture
- ReactorKit
- RxSwift, RxCocoa, RxDataSources
- SnapKit
- Moya
- Swinject
- Quick
- Nimble

## [ Develop Environment ]
- Language : Swift 5.X
- Xcode : 12.X
